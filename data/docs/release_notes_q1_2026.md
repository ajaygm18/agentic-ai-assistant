# Release Notes Q1 2026

## Release 3.4.0

Date: 2026-02-12

- Added workflow-level SLA timers visible in the support workspace.
- Introduced an escalation recommendation panel for Sev-1 and Sev-2 incidents.
- Fixed duplicate notification bugs for reassigned tickets.
- Improved analytics export speed by roughly 35 percent for monthly reports.

## Release 3.5.0

Date: 2026-03-18

- Added semantic knowledge search backed by Chroma and OpenAI embeddings.
- Introduced session-aware assistant memory for follow-up questions during a single support workflow.
- Added audit metadata to every generated support summary.
- Deprecated the legacy macro sidebar in favor of the new workflow composer.

## Known follow-up actions

- Teams rolling out 3.5.0 should retrain support agents on the workflow composer.
- Dashboard owners should update any saved views that depended on the legacy macro sidebar.
